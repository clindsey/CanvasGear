Basic Page Structure

The page will need to have the CanvasGear.js script included, a DOM node to bind the canvas to, CSS to set canvas dimensions, and an on_load callback to get things started once the page loads. 

      <!DOCTYPE html>
      <html>
        <head>
          <meta http-equiv="content-type" content="text/html;charset=utf-8">
          <title>Project Name</title>
          <script type="text/javascript" src="CanvasGear.js"></script>
          <script type="text/javascript">
            var on_load = function(){
            };
          </script>
          <style type="text/css">
            #canvas {
              width:320px;
              height:480px;
              margin:0 auto;
            }
          </style>
        </head>
        <body onload="on_load()">
          <div id="canvas"></div>
        </body>
      </html>
